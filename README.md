# -FreeCell-
这是一个基于C++、Qt实现的空当接龙小游戏，可供参考<br>
至于空当接龙的游戏规则，详见[FreeCell](https://en.wikipedia.org/wiki/FreeCell)<br>
<br>
文件 空当接龙 下是游戏程序源代码<br>
文件 CardGameServer 是记录程序排名的服务器<br>
